### Q1. 文件插入（100条记录，插为第10条）

#### (1) 连续分配 — 最少访问磁盘次数
前9条记录向前挪一个位置（向前挪比向后挪91条更省），操作：9次读 + 9次写 + 1次写新记录 = **19次**。

#### (2) FCB哪些内容改变
文件整体前移，起始块号变了；长度从100变101。**文件的首个盘块号和文件长度都要修改**。

#### (3) 隐式链接分配 — 最少访问磁盘次数
沿链读1~9块拿到原第10块指针：**9次读** → 分配新块写新记录：**1次写** → 修改第9块指针指向新块并写回：**1次写**。合计 **11次**。

#### (4) 文件系统支持的最大文件长度
每块512B，2B指针 → 有效数据510B/块。指针2B=16位 → 最多2^16个块。最大 = **510 × 2^16 B**。

### Q4. 索引分配（盘块1KB，索引项4B）

每个索引块存 1KB/4B = **256个指针**。

**(1) 3KB文件：** 数据块3个 + 索引块1个（256个位置，3项够用）= **4块**。

**(2) 3MB=3072KB文件：** 数据块3072个；一级索引块 ceil(3072/256)=12个；12个一级索引块需1个二级索引块管理。索引块共13个。总块 = 3072+13 = **(3K+13)块**。

### Q6. 分页32位地址，页大小4KB

**(1)** 页大小4KB = 2的12次方 → 页内偏移 **12位**；页号 = 32-12 = **20位**。

**(2)** 页表项数：2^20 = 1M项。每项4B → 最大页表 = **4MB**。

**(3)** 0000 13A5：页号=0x00001=1，块号=2 → 物理地址 = 2×4KB + 0x3A5 = **0000 23A5**。

### Q7. 分页32位，低12位偏移

**(1)** 12位偏移 → 页面大小 = 2的12次方 = **4096字节**。

**(2)** 1GB/4KB = 2^30/2^12 = **262144个物理页面**。

**(3)** 0000 248F：页号=2，物理块=7 → 物理地址 = 7×4096 + 0x48F = **0000 748F**。

---

### Q8. 页式16位地址，每页2KB，物理1MB

**(1)** 2KB = 2的11次方 → 偏移 **11位**。页号 = 16-11 = **5位**。
**(2)** 2^5 = **32页**，页表最多 **32项**。
**(3)** 物理块 = 1MB/2KB = 512个 → 需9位 → 至少 **2字节**（选项取近似值3）。
**(4)** 物理空间减半 → 块号从9位变8位 → **页表每项减少1位**。

### Q9. 非抢占与抢占调度

- **只能非抢占：FCFS**。进程主动放弃CPU才切换（等I/O或终止），没有时间片概念。
- **只能抢占：时间片轮转**。时间片到必须强制切换，不存在非抢占版本。
- 高优先权和短作业优先：既有抢占版本也有非抢占版本。

### Q14. 缓冲寄存器中断（9.6kb/s，32位寄存器）

**(1)** 32位填满：32/9600 = 1/300s = **3333μs**（选项取近似3200μs）。
**(2)** CPU响应窗口 = 下一位到达时间：1/9600s = **104μs**（选项取近似100μs）。
**(3)** 双缓冲后响应窗口从1位时间变为32位时间：**32倍**。

### Q19. char a[1024][1024]，1数据页(1KB)，按行存储

每页1024字节=刚好一行。数组1024行×1024列。

**按行访问 a[i][j]：** 一行在一页 → 1024次缺页。
**按列访问 a[j][i]：** 每次下一个元素跨一页 → 刚装入的页只用1字节就被淘汰 → 1024×1024 = **1048576次缺页**。

### Q21. 银行家算法（A=9,B=8,C=12，5进程）

**(1) T0时刻：** Need=Max−Allocation。Available=(9,8,12)−(8,7,11)=(1,1,1)。安全序列：P4(需1,0,1)→P1(2,1,0)→P2→P3→P5。**是安全状态**。

**(2) P1请求(0,1,0)：** 分配后Available=(1,0,1)，安全性检查通过。**可以分配**。

**(3) P2请求(1,0,0)：** Available=(0,0,1)，所有进程Need都>Available。**不能分配**。

## 三、核心公式速查

### Q22. 死锁避免
```
n进程各需w个同类资源：m ≥ n(w-1)+1
多进程各自需求不同：m ≥ Σ(每进程需求−1) + 1
例：3进程各需3,4,5：(3-1)+(4-1)+(5-1)+1=10
```

### Q23. 磁盘调度
```
FCFS：平均寻道最长。SSTF：最短但会饥饿
SCAN(电梯)：来回扫描。CSCAN：单向+跳回。FSCAN：分组防粘着
```

### Q24. 页面置换
```
OPT：未来最远淘汰(理论最优)。LRU：最近最久未用。FIFO：有Belady异常
```

### Q25. 页式地址转换
```
页号 = 逻辑地址 / 页大小；偏移 = 逻辑地址 % 页大小
物理地址 = 物理块号 × 页大小 + 偏移
```

### Q26. 文件系统
```
FAT大小 = 磁盘块数 × 表项大小
隐式链接有效数据/块 = 块大小 - 指针大小
```

### Q27. I/O与缓冲
```
中断间隔 = 缓冲寄存器位数 / 传输速率(bps)
DMA：块设备；程序中断：字符设备；通道：独立执行通道程序
```


## ✨ 特色

- **全章节覆盖** — 12 章完整知识笔记，每章 2-4 篇，重点突出
- **全景串联** — [`操作系统全景图.md`](操作系统全景图.md) 用「公司比喻」将 12 章串成一条线
- **300 题题库** — 选择题 + 填空题 + 判断题 + 连线题，分章节组织
- **考点速查** — 高频/中频/低频三级标注，适合期末突击
- **双向链接** — Obsidian `[[双链]]`，知识文件与题库互相引用
- **思维导图** — 调度算法、死锁与同步高频考点可视化

## 🗺️ 三条核心主线

| 主线 | 章节 | 核心问题 |
|------|------|----------|
| **CPU 管理线** | 第1→2→3→4章 | 进程怎么创建？CPU 先给谁用？怎么不打架？ |
| **内存管理线** | 第5→6→8→9章 | 内存怎么分？不够怎么办？数据怎么存磁盘？ |
| **I/O 与高级主题** | 第7→10→11→12章 | 怎么跟外设说话？多核/虚拟化/安全怎么做？ |

## 🚀 快速开始

1. **系统学习** — 从 `计算机操作系统 MOC.md` 进入，按章节顺序阅读
2. **期末冲刺** — 直接看 `附录/考点优先级速查.md`，锁定 ★★★ 考点
3. **刷题巩固** — `附录/` 下 6 个题库文件，按章节练习
4. **理解全景** — 读 `操作系统全景图.md`，用公司比喻建立直觉
